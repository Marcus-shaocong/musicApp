### 项目简介
这是一个使得用户可以浏览并操作歌曲和歌谱的小程序。

## 我们做了什么？
 - 主页
     + 封面图片轮转
     + 歌曲列表
         * 歌曲上拉刷新
         * 歌曲难易度
         * 歌曲收藏
         * 歌曲学习
     + 歌曲搜索 
 - 歌曲学习页面
     + 歌曲播放
     + 歌谱显示（非自动）
- 我的页面
    + 登陆 (开发中..)
    + 我的收藏 (开发中..)
    + 帮助中心 (开发中..)
    + 联系我们 (开发中..)
